# 天翼云 SK5 多 IP 搭建教程：实现矩阵独立 IP 多账号运营

## 1. 注册天翼云账号

首先，您需要拥有一个天翼云账号。如果您还没有注册，可以前往天翼云官网完成简单的注册流程。注册完成后，登录账号，为后续购买云服务器和配置多 IP 环境做好准备。

## 2. 购买云服务器

购买云服务器是搭建多 IP 环境的关键步骤。天翼云服务器分为**合营区**（天翼云与其他厂商合作的服务器）和**独营区**（天翼云独立运营的服务器）。以下是购买时需要注意的事项：

- **选择合营区服务器**：我们需要购买位于`合营区`的云服务器，因为这些区域支持更灵活的 IP 配置。
- **匹配 IP 与服务器地区**：购买的 IP 地区必须与云服务器地区一致。一台云主机最多可绑定 **10 个 IP**，但 IP 资源因地区而异，建议提前查看目标地区的 IP 库存情况，再决定购买。

### 合营区可选地区
以下是目前支持合营区的地区列表，供您参考：

- **华东地区**：上海4、浙江杭州、江苏苏州、江西南昌  
- **华南地区**：福建福州1、广东深圳、海南海口、湖北武汉2、广东广州4  
- **西北地区**：陕西西安2、新疆乌鲁木齐、甘肃兰州、青海西宁  
- **西南地区**：贵州、重庆、云南昆明  
- **北方地区**：山东青岛  

### 购买步骤
1. 登录天翼云控制台，进入服务器购买界面。
2. 根据需求选择合营区服务器，并确认 IP 可用性。
3. 完成支付，获取云服务器实例。

通过合理选择地区和服务器，您可以为多账号运营打下坚实基础，确保每个账号拥有独立的 IP 地址。

## 3. 添加弹性公网 IP

在购买云服务器后，您需要为服务器绑定额外的弹性公网 IP，以实现多 IP 管理。操作步骤如下：

1. 在云服务器管理界面，找到“弹性公网 IP”选项。
2. 根据需求申请新的 IP 地址，并将其绑定到服务器。

这一步骤能够有效扩展 IP 数量，满足矩阵式多账号运营的需求。借助多 IP 配置，您可以轻松管理多个独立账号，提高运营效率。

最后一步是将弹性子网 IP 与云服务器绑定，确保每个 IP 都能正常使用。具体操作如下：

1. 根据购买的弹性子网 IP，创建对应的子网。
